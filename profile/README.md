# Town & Country

## Repository format

The repositories starts with a abbreviation which defines what is inside of the repository.

| abbreviation | explanation |
| - | ---------------------- |
| A | Application Layer      |
| D | Domain Layer           |
| I | Infrastructure Layer   |
| S | Service                |
| F | Frontend               |
| T | Template               |
| E | Environment            |
| R | Rules Data             |
| M | Model ( Protobuffers ) |
| L | Library                |
